💍 Wedding-Ceremony

Welcome to the Wedding-Ceremony project repository.
This website is designed to provide a complete wedding event management experience. It helps guests explore and book services for different wedding functions including Mehndi, Mayo, Barat, and Valima along with catering services and event galleries.

✨ Features
🎉 Event planning for all wedding functions (Mayo, Mehndi, Barat, Valima)
🍽️ Professional catering service details
📸 Gallery showcasing past successful events
📅 Easy booking system for events
📱 Fully responsive design for all devices
🛠️ Technologies Used

This project is built using:

HTML5
CSS3
JavaScript
Bootstrap
